Set Color/Font for Selected

Saving ComparisonsYou can change the color and font of map components quite easily.

    • Select the component you want to change (i.e., the bar or label in a map)

Note: If you want to select multiple components, press and hold <Control> as you select them or, to select contiguous objects, click and drag the mouse to select them.

    • Select View >> Set color/font >> for selected

After a pause, the Display Dialog window appears.

You can set the colors for graphic objects and change the font, including size and color, here.

    • Define the color you want for the graphics

For help with the specifics of color definition, see Using the Color Definition Methods.

    • In the Font attributes section, select the font option (the top button in the section)

A list of fonts pops up.

    • Select the font you want from the list

Look at the Bold and Italic options. Checked options are in effect.

    • Select the box to the left of either option to check or uncheck it
    • Select the number option

A list of font sizes pops up.

    • Select the size you want for your text
    • Select Apply
    • If you want to change other graphic components, select them
    • Repeat these steps until you've defined all of the graphic and text characteristics you want
    • Select OK

The window closes.
