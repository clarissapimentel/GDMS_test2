Accessing the Viewer

To access the Comparative Map Viewer, you must first launch ISYS. If you don't know how to do this, talk to your system administrator.

The ISYS window lists all of the available ISYS tools.

    • Select the NCGR/CGIAR Comparative Map Viewer button

      OR

    • Select Services >> NCGR/CGIAR Comparative Map Viewer
