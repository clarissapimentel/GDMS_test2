Using Zoom

Zoom options are available only in the individual Map windows. When maps initially appear, the entire map is shown, i.e., the display is zoomed out completely. At this resolution you may not be able to see many of the details in the map. The zoom options let you adjust the display to the resolution that is most useful for you.

To see the greatest detail

    • Select View >> Zoom max

To see the entire map in one window

    • Select View >> Zoom all

To see a little more detail in your map

    • Select View >> Zoom in

You can repeat this until you get the detail you want.

To see more of the map in the window

    • Select View >> Zoom out

You can repeat this until you have what you want in the display.
