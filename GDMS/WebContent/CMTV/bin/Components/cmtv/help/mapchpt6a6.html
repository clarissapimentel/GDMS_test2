Viewing Comparisons Graphically

When you compare maps, a graphical representation of the comparison is automatically displayed. You can clear the Viewer pane, though. If you've done this, you can choose to have comparison(s) redisplayed.

    • Select the comparison(s) in the Map Comparisons pane

      Note: If you want to select more than one comparison, press and hold <Control> as you select the comparisons.

    • Select View graphically

The comparison appears in the Viewer pane.
