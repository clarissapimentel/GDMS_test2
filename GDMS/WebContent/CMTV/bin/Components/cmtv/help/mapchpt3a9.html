File Menu

The File menu in the Map Viewer window has five options:

  • Load

    Lets you select a file and load it. For more information about this, see Loading Map Data from a Flatfile and Loading Map Data from a Database.

  • Save all to XML

    Lets you save all of the maps and comparisons you have in the Viewer to a single XML file. For more information about this, see Saving Your Maps.

  • Clear desktop

    Lets you remove all of the maps from the Map Viewer pane.

  • Print desktop display

    Lets you send the graphical contents of the Map Viewer pane to a printer or to a file. For more information about this, see Printing Maps.

  • Exit

    Lets you close the Comparative Map Viewer.
