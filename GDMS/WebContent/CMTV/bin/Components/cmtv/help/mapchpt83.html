Using the DynamicDiscovery Dialog

The DynamicDiscovery Dialog is always accessible from the DynamicDiscovery menu.

The window provides a list of the ISYS services that are available for the selected data type. The top list is the Viewer services--services you can use to look at the selected data in another way. The bottom list is Data services--services that will query for related data.

    • Select one or more of the services
    • If you want the window to stay open so you can execute another service, select the box to the left of Stay open after executing services
    • Select Execute

The selected services are started. The steps that follow will depend on the services you selected.
