Defining the Graphical Properties of Your Maps

There are three ways to define the graphical properties of your maps.

  • You can select which objects will appear in your maps with the Customize Layout option

    You may choose to show only graphical bars or perhaps only the labels for genes. For more information about this option, see Customize Layout.

  • You can change the colors and fonts used for selected items in selected maps with the Set Color/Font for Selected option

    For more information about this option, see Set Color/Font for Selected.

  • You can select to load and use a pre-defined rule and/or define the colors and fonts to be used for all maps with the Set Color/Font for User-Defined Rule option

    These rules may define the colors to be used for each object, the graphic symbol to be used for particular objects, etc. When you change rule selection, the currently displayed maps are immediately redisplayed to conform to the selected rules.

    For more information about these options, see Set Color/Font for User-Defined Rule and Loading and Selecting Rules for Map Display.

The View menu options let you set the graphical properties for your maps.
