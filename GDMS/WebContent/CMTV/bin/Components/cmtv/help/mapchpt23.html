Load a Map

Start with a new, empty Comparative Mapping Viewer.

    • Select File >> Load >> Load from ICIS map output file

The Open window appears.

    • Select IcisMaps
    • Select Open

The maps in the selected file are listed in the Maps Currently Available pane. You will have to adjust the pane size to see the full map names.

    • Move the mouse cursor to the "textured" divider at the right of the pane
    • When the pointer becomes a line with an arrow pointing to it, click and drag the mouse to make the pane wider

    • Select CT9993xIR62266:1
    • Select View separately

A graphical map appears in the Viewer pane.

The map is within its own Map window in the pane.
