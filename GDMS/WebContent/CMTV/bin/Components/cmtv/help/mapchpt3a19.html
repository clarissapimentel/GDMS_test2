View Menu

The View menu in the individual maps has five options.

  • Switch display mode

    Lets you change the orientation of the map from vertical to horizontal or from horizontal to vertical. For more information about this option, see Displaying Graphical Maps.

  • Zoom all

    Lets you see the entire map. When maps are first displayed they are zoomed out by default.

  • Zoom out

    Lets you see more of the map than is currently shown. This option only works if the entire map is not already visible.

  • Zoom in

    Lets you focus in and see more detail in the map.

  • Zoom max

    Lets you see the map in the greatest magnification possible.

For more information about the zoom options, see Using Zoom.
