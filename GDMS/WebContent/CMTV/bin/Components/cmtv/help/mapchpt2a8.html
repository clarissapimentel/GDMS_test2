Example Three

We'll start this example with an newly opened Viewer.

    • Select File >> Load >> Load from GenBank flatfile

The Open window appears.

    • Select file U26846.gb
    • Select Open

A map is added to the Maps Currently Available pane.

    • Select View >> Customize layout

The Layout Dialog window appears.

    • Make sure all label options are unchecked.
    • Select Set as default
    • Select OK
    • Select View >> Load IsysAttribute-based property assignments from file
    • Select file graphical_properties.sample
    • Select Open
    • Select View
    • Select the box to the left of From file graphical_properties.sample: Assign colors based on type attribute
    • Select the map in the Maps Currently Available pane
    • Select View separately

The map appears in the Viewer pane.

    • Press the right mouse button in the map

The DynamicDiscovery menu appears.

    • Select NCBI BLAST

The Similarity Search Launcher window appears.

    • Select the box to the left of drosoph
    • Select Options
    • Type in .001 for Return option
    • Select Use These Values
    • Select Launch Query

After a while, the Similarity Search Browser window appears with results.

    • Select the record (AE003644.2)
    • Press the right mouse button
    • Select NCGR/CGIAR Comparative Map Viewer

The map appears in the Viewer pane.

Move it so you can see both maps.

    • In the Map Comparison pane, select the comparison and View graphically

The comparison is illustrated.

    • Select both maps
    • Select View >> Customize layout
    • Make sure that the only label turned on is Type
    • Select Apply to selected
    • Select Set as Default
    • Select OK

Notice that each object in AE003644.2 corresponds to two locations in 862898. One case appears to be gene duplication. The other appears to be a repeat.

The representation of the background sequence that we received from BLAST contains little information about it other than the representation of those regions that matched our query sequence. However, since we have an identifier for this sequence, we can retrieve an annotated version of the same sequence from Entrez. Finally, we can combine the two perspectives on the same sequence into a single view and see how the matched regions of the sequence correspond to the annotated regions.

    • Select the map for AE003644.2
    • Press the right mouse button
    • Select Retrieve from Entrez: Retrieve from Entrez (annotated): AE003644.2

The DynamicDiscovery Dialog appears.

    • Select NCGR/CGIAR Comparative Map Viewer
    • Select Execute

The annotated map appears.

    • In the Maps Currently Available pane, select the BLAST map (AE003644.2)
    • Press and hold <Control> as you select the annotated map (10728786)
    • Select View together

A combined view appears.

    • Click and drag to select all of the objects in the area where the BLAST map objects appear (the 100,500 area)
    • Press the right mouse button
    • Select Display in table

As you can see, the region that matched has gene annotations.

Now, let's compare these mapped objects with those in our original sequence.

    • In 862898, press and hold <Control> as you click and drag to select the markers around the BLAST markers
    • Press the right mouse button
    • Select Display in table

Another Table Viewer window appears and you can examine the mapped objects.
