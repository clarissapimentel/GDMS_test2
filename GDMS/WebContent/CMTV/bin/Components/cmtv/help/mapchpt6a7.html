Viewing Comparisons Tabularly

Although the graphical maps illustrate the correspondences in compared maps, you may find it helpful to see the correspondences summarized in tabular form.

    • Select the comparison(s) in the Map Comparisons pane

      Note: If you want to select more than one comparison, press and hold <Control> as you select the comparisons.

    • Select View tabular

The Map Comparisons window appears.

This window contains one or more sections with tables that show the corresponding markers in the comparison(s). Each section has a Comparison Mapping Tools menu with options you can use to build consensus maps. For more information about this menu and consensus maps, see Building Consensus Maps.

The Map Comparisons window is synchronized with the Viewer window. So, when you select markers in the tabular view, the markers are also selected in the graphical map in the Viewer pane.

You can add comparisons to the window or remove all of the comparisons from it. Once you've viewed a comparison tabularly, it remains in the Map Comparisons window until you reset the window.

To add a comparison to the Map Comparisons window

    • Select the comparison(s) in the Map Comparisons pane in the Viewer window

      Note: If you want to select more than one comparison, press and hold <Control> as you select the comparisons.

    • Select View tabular

The comparison is added to the Map Comparison window.

To remove all comparisons from the Map Comparisons window

    • Select Reset

All comparisons are removed from the Map Comparisons window but the window remains open.

To close the Map Comparisons window

    • Select Close
