Using the NCGR/CGIAR Comparative Map Viewer Window

When you access the Comparative Map Viewer the NCGR/CGIAR Comparative Map Viewer window appears.

This window has three panes:

  • Maps Currently Available Pane in the upper left corner
  • Map Comparisons Pane in the lower left corner
  • Map Viewer Pane on the right side

When data are loaded, the contents of these panes are synchronized. So, if you select a map in the Map Viewer pane, that map will be highlighted in the Maps Currently Available pane. You can resize the panes, resize and move windows, and adjust column width to make data more visible. For help with these tasks, see To resize panes, To resize a window, To move a window, and To change column width.

In the menu bar, just below the menu title, you'll find five menus.

  • File Menu
  • Edit Menu
  • View Menu
  • Filter Menu
  • Map Comparison Menu

To open a menu

    • Point to or select the menu name

The menu will open below the name and you can select options from it.

To close a menu

    • Select the menu name again

      OR

    • Select elsewhere in the menu bar
