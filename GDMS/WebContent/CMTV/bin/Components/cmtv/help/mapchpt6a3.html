Letting the Viewer Find Comparable Maps

There are two menu options you can use to have the Viewer find comparable maps. The first is on the Map Comparison menu in the Viewer window. You can apply this option to one or more maps. The second is on the Map Comparison menu in the Map windows. This option applies only to the map in the window.

To find comparable maps for a single map

    • Load the map data you want the Viewer to search for comparisons
    • In the Map window for the map you want to compare, select Map comparison >> Find comparable maps

      OR

    • Select a map in the Viewer or Maps Currently Available pane
    • Select Map comparison >> Find maps comparable to selected maps

The Map Comparisons dialog appears.

    • Select the drop-down menu button and select a minimum number
    • Select OK

The comparison is generated and the panes are updated with comparison information. If multiple comparable maps are found, only the first comparison is shown in the Viewer pane. You can view the other comparisons using the View Graphically button in the Map Comparisons pane. For help with this, see Viewing Comparisons Graphically.

The comparison is generated using the default algorithm or, if you selected an algorithm, your selection. For more information about selecting an algorithm, see Selecting a Correspondence Algorithm for Your Comparisons.

To find comparable maps for multiple maps

    • Load the map data you want the Viewer to search for comparisons
    • In the Viewer or the Maps Currently Available pane, select the maps you want compared
    • In the Viewer window, select Map comparison >> Find maps comparable to selected maps

The Map Comparisons dialog appears.

    • Select the drop-down menu button and select the number you want
    • Select OK

The comparison is generated and the panes are updated with comparison information. If more than one comparable map is found, only the first is shown in the Viewer pane. You can view the other comparisons using the View Graphically button in the Map Comparisons pane. For help with this, see Viewing Comparisons Graphically.

The comparison is generated using the default algorithm or, if you selected an algorithm, your selection. For more information about selecting an algorithm, see Selecting a Correspondence Algorithm for Your Comparisons.
