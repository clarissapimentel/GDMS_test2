Loading and Selecting Rules for Map Display

When you define a color/font assignment rule, an option is added to the View menu. You can select to apply these rules or not using the selection boxes to the left of the options.

You can also load existing, pre-defined rules. One sample rule is provided in file graphical_properties.sample. This rule defines colors for different classes of objects:
Object Color
CDS green
STS magenta
TATA_signal yellow
exon green
gene cyan
mRNA orange
mat_peptide green
misc_difference orange
misc_feature orange
modified_base orange
polyA_signal yellow
polyA_site yellow
protein_bind green
rRNA cyan
repeat_region pink
repeat_unit pink
sig_peptide green
source darkGray
stem_loop white
tRNA cyan

To load a rule

    • Select View >> the load option

To apply a rule

    • Select View
    • Select the box to the left of the rule(s) you want to apply

You can apply more than one rule. The last rule applied may override the definitions in previous rules.
