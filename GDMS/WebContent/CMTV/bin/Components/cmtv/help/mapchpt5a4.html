Selecting and Deselecting Maps

You will need to select graphical maps to perform many tasks. When a map is selected, the title bar is colored blue. A map that is not selected has a gray title bar.

To select a map

    • Select the Map window's title bar or the window's border

      OR

    • Select the map in the Maps Currently Available pane

The Map window's title bar is colored blue.

To select multiple maps

    • Press and hold <Control>
    • Select the title bar or window borders of the maps you want

      OR

    • Select the maps in the Maps Currently Available pane

The title bar of each selected Map window is colored blue.

To select all maps

    • Select Edit >> Select all

All maps and all map contents are selected.

To select all map objects in a map

    • In the Map window, select Edit >> Select all

All object in the map are selected.

To deselect a map

    • Press <Control> as you select the map title bar

The Map window's title bar is colored gray.

To invert the selection of all maps

    • Select Edit >> Invert selection

All maps and map components that were previously selected are deselected while those that were not selected are selected.

To invert the selection in one map

    • In the Map window, select Edit >> Invert selection

All objects in the map that were previously selected are deselected while all others are selected.
