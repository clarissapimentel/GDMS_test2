About the Comparative Map and Trait Viewer

As an ISYS component, the Comparative Map and Trait Viewer can provide access to a variety of databases and other tools. The Viewer itself displays maps, mapped objects, and regions of correspondence between maps. The Viewer can make graphical representations of any data that describe objects in a one-dimensional coordinate system that has minimum and maximum coordinate values. For example, a GenBank flatfile defines a one-dimensional coordinate system.

An ICIS map output file also defines a one-dimensional coordinate system.

The Comparative Map and Trait Viewer translates coordinates like these into graphical representations, including mapped objects, which may be genes, RFLPs, QTLs, etc. The GenBank file is mapped on the left, below; the ICIS file on the right.

Each map is presented in an independently-controlled view. The Viewer provides options you can use to filter the map view, zoom in and out, and change the map orientation. You can select to change the visual representation of maps by changing the colors of particular elements, changing the appearance of particular labels, or segregating objects by type. In addition to the options available in the Viewer, visual representation can be changed with plug-in visualization components in ISYS.

You can also, of course, use the Viewer to compare two or more maps and see the comparison illustrated. You can use the Viewer to find comparable maps, print maps, and build and save consensus maps. In the following example, the maps of two ICIS records are compared. Lines connect the corresponding objects on the maps.

Finally, since the Viewer is part of the ISYS platform, you can use DynamicDiscovery to explore other possibilities in ISYS. For example, if you select several markers in a map and press the right mouse button, a DynamicDiscovery menu appears. Depending on your ISYS configuration, this menu might include options to query other data sources (e.g., NCBI's Entrez, ICIS, IWIS, etc.) for related data or get allele values for markers/germplasms.
