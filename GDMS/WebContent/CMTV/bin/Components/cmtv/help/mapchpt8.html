8 Using DynamicDiscovery

DynamicDiscovery is a patent pending feature of ISYS that links the functionality and services of all ISYS components. When programs are "plugged into" ISYS, they tell ISYS the types of data they use. When you select data in a component, then, and invoke DynamicDiscovery, ISYS returns a list of the options in all ISYS components that are relevant for that data type.

When you invoke DynamicDiscovery in the Comparative Map Viewer, you may be presented with options to display the selected data in a table, start a web-based query, get allele values for markers/germplasms, or query or retrieve other related data from databases. The options will vary depending on the components in your ISYS system.
