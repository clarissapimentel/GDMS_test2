Filtering Maps

When graphical maps are displayed, there are multiple Filter menus--one in the Map Viewer window and one in each Map window. The Filter menu options in the Map Viewer window apply to all displayed maps. The options in a Map window apply only to that map. There are three filtering options:

  • Hide selected mapped objects

    Lets you remove selected objects from the screen.

  • Hide unselected mapped objects

    Lets you remove all objects except those you've selected from the screen.

  • Show all hidden mapped objects

    Restores all previously hidden objects to the screen.

To hide map objects in a single map

    • Select the object(s) you want to hide

      Note: There are three ways to select multiple objects: 1) Click and drag to select adjacent objects; 2) Select the map at the beginning or end of a contiguous block, press and hold <Shift>, and select the map at the other end of the block; 3) Press and hold <Control> as you select objects.

In the Map Viewer window or in the Map window

    • Select Filter >> Hide selected mapped objects

      OR

    • Select Filter >> Hide unselected mapped objects

To hide objects in multiple maps

    • Select the object(s) you want to hide

      Note: There are three ways to select multiple objects: 1) Click and drag to select adjacent objects; 2) Select the map at the beginning or end of a contiguous block, press and hold <Shift>, and select the map at the other end of the block; 3) Press and hold <Control> as you select objects.

In the Map Viewer window

    • Select Filter >> Hide selected mapped objects

      OR

    • Select Filter >> Hide unselected mapped objects

To restore all objects to a map

    • In the Map window, select Filter >> Show all hidden mapped objects

The previously hidden objects appear in the map.

To restore all objects to all maps

    • In the Map Viewer window, select Filter >> Show all hidden mapped objects

The previously hidden objects appear in all maps.
