Annotating Maps

You can enter short notes to annotate the maps in the Viewer. You can also view and edit existing annotations.

To enter an annotation

    • Select one or more markers in a graphic map
    • Press the right mouse button

      Note: Remember that if you press the right mouse button while pointing to a particular map, the annotations you enter will apply only to the selected objects in that map; if you press the right mouse button while pointing to the desktop, the annotations you enter will apply to all of the selected objects in all of the maps in the desktop.

The DynamicDiscovery menu appears.

Your menu will vary depending on your system.

    • Select Add annotation for selected map objects

The Enter Annotation Text dialog appears.

    • Type in your comment
    • Select OK

The dialog closes and the annotation is stored. To save the annotation beyond this session, though, you must save the map. For help with this, see Saving Your Maps.

To view annotations

    • Select one or more markers in a graphic map
    • Press the right mouse button

The DynamicDiscovery menu appears.

    • Select Show annotations for selected map objects

The Annotations for Map Objects window appears with a list of the annotations and information on who entered them and when.

If the annotation is too long to be seen in the table column

    • Move your mouse cursor to the annotation

A pop-up box appears with the full annotation text in it.

To edit annotations

In the Annotations for Map Objects window

    • Point to the annotation you want to change and press the right mouse button
    • Select Edit annotation

The Edit Annotation Text dialog appears.

    • Change the annotation as you wish
    • Select OK
