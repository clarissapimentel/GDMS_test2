Options Menu

The Options menu in the Table Viewer has two options:

  • Reset column widths

    Lets you return the columns to their original width after you've changed them. You may need to change the width of some columns to see all of the data in them. For help with this, see Adjusting Column Width.

  • Fit columns to width

    Lets you have the Viewer adjust the width of all columns to fill the window. When this option is selected, the column widths are adjusted whenever you adjust the window size.

To reset columns to their default width

    • Select Options >> Reset column widths

To adjust columns to fill the window

    • Select Options
    • Select the box to the left of Fit columns to width
