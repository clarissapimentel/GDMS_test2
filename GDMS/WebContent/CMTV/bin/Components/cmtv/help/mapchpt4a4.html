Saving Map Data

You can save map data for one or more maps in a single XML file using the Save option in the Maps Currently Available pane.

    • In the Maps Currently Available pane, select the map(s) you want to save

      Note: To select multiple maps, press and hold <Control> while you select them. Or, to select contiguous maps, click and drag the mouse cursor over the maps or select the map at the beginning or end of the contiguous block, press and hold <Shift>, and select the map at the other end of the block.

    • Select Save

The Save window appears.

This window shows the files that are in the directory shown at the top. The navigation buttons at the top are identical to those in the Open window. For more information about them, see Loading Map Data from a Flatfile.

    • Select a file you want to overwrite or type in a new file name
    • Select Save

The file is saved in the directory shown in the window.
