Displaying Graphical Maps

To display graphical maps, you must first, of course, have map data loaded into the Viewer. When map data are loaded, maps are listed in the Maps Currently Available pane.

When a map first appears, the entire map is shown in a vertical orientation. You can change the orientation using the View menu in the Map window.

Maps are displayed with graphical characteristics as defined in your graphical properties definition. For more information about changing the graphical display, see Defining the Graphical Properties of Your Maps.

To load a map

    • Select the map in the Maps Currently Available pane
    • Select View Separately

The map appears in the Viewer pane.

To load multiple maps

    • Press and hold <Control> as you select the maps in the Maps Currently Available pane

      OR

    • To select contiguous maps, click and drag the mouse cursor over the maps

      OR

    • To select contiguous maps, select the map at the beginning or end of the contiguous block, press and hold <Shift>, and select the map at the other end of the block
    • Select View separately

The maps appear in the Viewer pane. To see them you will need to move the map windows. Also, you may need to adjust the size of the Viewer pane and the map windows. For help with these tasks, see Moving Windows, Resizing Panes, and Resizing Windows.

To change map orientation

    • In the Map window, select View >> Switch display mode

The map is redisplayed in a new orientation (horizontal or vertical).

To minimize the Map window

    • Select the minimize icon in the title bar:

The window is reduced to a rectangle at the bottom of the desktop.

To open a minimized Map window

    • Select the map rectangle in the desktop

To maximize the Map window

    • Select the maximize icon in the title bar:

The map window is increased to fill the desktop.

To reduce a maximized Map window

    • Select the reduce icon (which replaced the maximize icon when the window was maximized)
