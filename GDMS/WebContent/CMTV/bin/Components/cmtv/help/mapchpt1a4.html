Conventions

The following conventions were used in writing this help.

Commands

Instructions telling you to take some action will usually appear as in the following:

    • Enter the identifier

Enter in such instructions means that you should type in something and then press the <Return> key. If the instruction says to type something, do not press <Return>.

Using the Mouse

When you are instructed to select something, use the mouse to move the pointer (arrow, crosshairs, or hand) on your screen to the item you are selecting. Then, press the left mouse button to click once on the selection.

Instructions telling you to select two or more consecutive items will have selections separated with ">>" as in the following:

    • Select File >> Save

If you are instructed to click and drag something, use the mouse to move the pointer to the item you are selecting and then press and hold the left mouse button while you move the mouse to the new location.
