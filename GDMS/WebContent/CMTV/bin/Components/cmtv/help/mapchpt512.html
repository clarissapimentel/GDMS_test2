Set Color/Font for User-Defined Rule

Each time you use the Viewer, you can define a general rule for displaying map contents. The rule is kept for the duration of your session and may be applied to the maps in the Viewer at your discretion. The rule is not, however, saved when you exit the Viewer.

To define a rule

    • Select View >> Set color/font >> for user-defined rule

The Display Dialog window for defining rules appears.

Here you can define the colors and fonts for all map objects.

    • Select one or more classes of objects in the Annotations list on the left

      Note: If you want to define the same characteristics for two or more objects, press and hold <Control> while you select them in the Annotations list.

    • Set the color you want for the object(s)

For help with the specifics of color definition, see Using the Color Definition Methods.

    • In the Font attributes section, select the font option (the top button in the section)

A list of fonts pops up.

    • Select the font you want from the list

Look at the Bold and Italic options. Checked options are in effect.

    • Select the box to the left of either option to check or uncheck it
    • Select the number option

A list of font sizes pops up.

    • Select the size you want for your text
    • Select Apply
    • Repeat these steps until you've defined rules for all of the objects you want
    • Select OK
