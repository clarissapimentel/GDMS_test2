7 Building Consensus Maps

You can build consensus maps from two or more maps in the Comparative Map Viewer. To be useful, the maps you use to build a consensus map must share many markers. Generally, these maps will represent the same chromosome of a particular species. To build a consensus, you must identify common markers as anchors. Then you can have the Viewer build a consensus map based on those anchors.

Currently, the Viewer provides only one method of positioning anchors on a consensus map. This method begins by taking an average of the anchor positions on the input maps. Then it calculates positioning of non-anchor markers and adjusts the positioning of all markers, including the anchors, to ensure that all markers can be placed on the consensus map. This adjustment means that the anchors are not placed in a truly average position.

The consensus mapping process begins with a comparison.

    • Load the maps

For help with this, see Loading Map Data.

    • Select the maps
    • Generate a map comparison

For help with this, see Comparing Maps.

    • Select the comparison in the Map Comparisons pane
    • Select View tabular

The Map Comparisons window appears with a tabular presentation of the comparable markers in the maps.

The first step in consensus mapping is to select anchor markers. You can do this in three ways:

  • Let the Viewer select anchors for you
  • Select your own anchors
  • Let the Viewer select anchors and then add others

You can do all of these using the Consensus Mapping Tools menu in the Map Comparisons window and the maps in the Viewer window. Remember, though, that

  • Anchors must be common to all of the maps
  • You must have at least two anchors to build a consensus map

To let the Viewer select anchors for you

    • Select Consensus mapping tools >> Select anchor markers >> Select well-ordered markers common to all input maps

The Viewer selects markers. These suggested anchors are highlighted in the Viewer window and in the Map Comparisons window.

To select your own anchors

    • Press and hold <Control> as you select the markers you want as anchors

When the anchors you want are highlighted via either method

    • Select Consensus mapping tools >> Create consensus map with selected markers as anchors

A new comparison is listed in the Map Comparisons pane and a consensus map is added to the Viewer window. You may need to move it to see it and the other maps clearly.

Now you can add the other markers to the consensus map. The Map window containing the consensus map has a Marker Manipulations menu that you can use to adjust the markers in the map.

    • Select Marker Manipulations >> Position non-anchor markers

The consensus map is updated. Depending on how many markers were in your original maps, the consensus map may be very crowded. You may find it helpful to hide the map comparison lines. You may also want to merge some markers and remove some markers from the consensus map.

To hide the map comparison lines

    • In the main Viewer window, select Map comparison >> Hide all map comparisons

To merge selected markers

    • Press and hold <Control> as you select the corresponding markers you want to merge on the consensus
    • In the Consensus Map window, select Marker manipulations >> Merge selected markers

To have the Viewer merge corresponding markers

    • In the Consensus Map window, select Marker manipulations >> Merge corresponding markers that are closely positioned

The Marker Merge dialog appears.

    • Type in the maximum distance you want
    • Select OK

The Viewer merges any corresponding markers that are the distance you entered apart or closer.

To remove markers

    • Select the marker(s) you want to remove

      Note: To select multiple markers, press and hold <Control> while you select the markers or click and drag to select contiguous markers.

    • Select Marker manipulations >> Remove selected markers

The markers are removed from the consensus map.
