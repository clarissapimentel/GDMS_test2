Loading Map Data from a Flatfile

Most often you will be loading map data from a flatfile. Be sure you know the format of the file you want to load before you start.

    • Select File >> Load

The Load submenu appears.

This menu may vary depending on your ISYS configuration.

    • Select the flatfile format option you want

The Open Window appears, showing the contents of the data directory.

You can use the options at the top to move to another directory if you wish.
Options Description

This button shows the name of the current directory. The button accesses a drop-down menu that lists all of the directories above the current one. You can select any directory in the menu to access its contents.

    • In the directory you want, select the file you want.

The filename appears in the File name field at the bottom of the window.

    • Select Open

The file contents are loaded into the Viewer and listed in the Maps Currently Available pane. If the file you loaded contains comparisons, the comparisons are listed in the Map Comparisons pane.

You can load as many maps as you wish, within the limits of your system.
