Map Comparison Menu

The Map Comparison menu in the Map Viewer window has five options:

  • Generate map comparisons for selected maps

    Lets you have the Viewer compare maps. For more information about this option, see Having the Viewer Compare Selected Maps.

  • View all comparisons between selected maps

    Lets you view a graphical illustration of comparisons. For more information about this option, see Hiding and Unhiding Comparisons.

  • Find maps comparable to selected maps

    Lets you have the Viewer search for comparable maps. For more information about this option, see Letting the Viewer Find Comparable Maps.

  • Hide all map comparisons

    Lets you remove comparisons from the Map Viewer pane. For more information about this option, see Hiding and Unhiding Comparisons.

  • Set correspondence algorithm

    Lets you set the default algorithm for map comparison. For more information about this option, see Selecting a Correspondence Algorithm for Your Comparisons.
