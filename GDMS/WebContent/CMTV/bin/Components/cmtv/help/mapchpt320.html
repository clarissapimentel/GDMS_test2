Filter Menu

The Filter menu in the individual maps has three options that let you quickly change which markers are shown in the map.

  • Hide selected mapped objects

    Lets you remove all selected objects from the maps in the Map window.

  • Hide unselected mapped objects

    Lets you remove all of the objects that are not selected in the Map window.

  • Show all hidden mapped objects

    Lets you restore all previously hidden objects in the Map window.

For more information about these options, see Filtering Maps.
