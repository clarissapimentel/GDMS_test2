Viewing Graphical Maps

    • Select one or more maps in the Maps Currently Available pane

      Note: To select more than one map, press and hold <Control> as you select the maps. Or, to select contiguous maps, click and drag the mouse cursor over the maps or select the map at the beginning or end of the contiguous block, press and hold <Shift>, and select the map at the other end of the block.

    • Select View separately

The maps you selected appear in graphical format in the right pane--the Map Viewer pane. If you selected more than one map, you'll have to move the Map windows around in the pane to see them.

For more information about graphical maps and the options associated with them, see Working With Graphical Maps.
