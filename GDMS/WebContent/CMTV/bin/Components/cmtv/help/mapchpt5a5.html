Removing Graphic Maps from the Window

To remove all of the maps from the pane

    • Select File >> Clear desktop

All of the graphical maps are removed from the Map Viewer pane.

To remove a single map from the pane

    • Select the boxed "x" in the top right corner of the Map window.

The selected Map window closes.
