Loading Map Data

    • Select File >> Load >> the load option that matches your data
    • Select the file containing your map data
    • Select Open

The maps in your file are loaded into the Viewer and are listed in the upper left pane--the Maps Currently Available pane.

For more details about this step and the options in the Maps Currently Available pane, see Loading Map Data.
