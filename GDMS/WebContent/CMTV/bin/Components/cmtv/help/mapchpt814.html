View Menu

The View menu has two options.

  • Set Id Label Size

    Lets you select the size of the ID label text in the current view.

  • Set Data Label Size

    Lets you select the size of the data label text in the current view.

To change the size of labels

    • Select View >> the label option you want

The Select Label Size dialog appears.

    • Select the numbered button

A short list of sizes appears. You can select the up and down arrowheads to the right of the list to see more size options.

    • Select the size you want from the menu
    • Select OK

The labels you selected to change are resized.
