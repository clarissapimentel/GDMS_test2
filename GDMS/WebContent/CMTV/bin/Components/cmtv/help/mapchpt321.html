Map Comparison Menu

The Map Comparison menu in the individual maps has three options.

  • Find comparable maps

    Lets you search the loaded maps to find all maps that can be compared to this map. For more information about this option, see Letting the Viewer Find Comparable Maps.

  • Find combinable maps in memory

    Lets you search the maps in memory to find all maps that can be compared to this map.

  • Find combinable maps from repository

    Lets you search repository maps to find all maps that can be compared to this map.
