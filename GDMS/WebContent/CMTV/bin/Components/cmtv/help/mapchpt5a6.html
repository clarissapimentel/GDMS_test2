Printing Maps

You can print one or more maps from the Viewer. When you print, the map or maps are printed exactly as they appear on the screen.

To print a map

    • Adjust the map on the screen so that it appears the way you want it printed
    • In the Map window, select File >> Print

The Print window appears.

This window lets you define how many copies you want printed, the name of the printer, any title you want printed, and any printer command options you want used. You can also choose to print the map to a postscript file and define the filename.

    • Select and/or type in the options you want
    • Select Print

To print the Map Viewer pane contents

    • Adjust the maps on the screen so that they appear the way you want them printed
    • In the main window, select File >> Print desktop display

The Print window appears.

    • Select and/or type in the options you want
    • Select Print
