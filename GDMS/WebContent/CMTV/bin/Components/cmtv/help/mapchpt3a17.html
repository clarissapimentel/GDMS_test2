File Menu

The File menu in the individual maps has two options:

  • Save

    Lets you save the map, including any annotations you've added. For more information about this option, see Saving Your Maps.

  • Print

    Sends the map to a printer or prints it to a postscript file. For more information about this option, see Printing Maps.
