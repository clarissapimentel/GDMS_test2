File Menu

The File menu in the Graph Viewer has three options:

  • Open

    Lets you load a graph. This option is only available if there is not already a graph in the window.

  • Clear

    Lets you remove the current graph from the Graph Viewer without closing the Viewer.

  • Quit

    Lets you exit the Graph Viewer.

To open a new graph

    • Select File >> Open

The Open window appears. For help with the navigation options in this window, see Loading Map Data from a Flatfile.

    • Select the file you want to open
    • Select Open

To clear the Graph Viewer window

    • Select File >> Clear

The window is cleared.

To exit the Graph Viewer

    • Select File >> Quit

      OR

    • Select Quit (at the bottom of the window)
