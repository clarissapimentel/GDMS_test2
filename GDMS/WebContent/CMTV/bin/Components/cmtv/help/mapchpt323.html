Marker Manipulations Menu

The Marker Manipulations menu appears only in consensus map windows. It has four options.

  • Position non-anchor markers

    Lets you place all of the non-anchor markers on the consensus map.

  • Merge selected markers

    Lets you merge corresponding markers in the consensus map.

  • Remove selected markers

    Lets you erase markers from the consensus map.

  • Merge corresponding markers that are closely positioned

    Lets you have the Viewer merge corresponding makers that are closely positioned in the consensus map.

For more information about these options, see Building Consensus Maps.
