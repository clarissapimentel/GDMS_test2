Using the ISYS Graph Viewer

The ISYS Graph Viewer is one of the Viewer services in ISYS. It presents data in a graphical node and arrow format.

When the Graphic Viewer initially appears, the entire graphic is shown in the window. Often, at this level of resolution, you will not be able to see the nodes and their labels. You can use the zoom scale at the bottom to see more details.

There are three menus at the top of the ISYS Graph Viewer window:

  • File
  • View
  • Layout

There are two buttons at the bottom of the window:

  • Size to Fit
  • Quit
