Maps Currently Available Pane

This pane is in the top left corner of the Viewer window.

This pane lists all of the map data you have loaded into the Viewer during the session. These data are listed in two columns.
Data Description
  • Map
The map data identifier; may also include a title or description of the map content
  • Mapped object count
The number of mapped objects in the dataset

At the bottom of the pane are four buttons:

  • View separately

    Lets you have selected maps appear as graphics in the Map Viewer pane. For more information about this option, see Displaying Graphical Maps.

  • View together

    Lets you have selected maps combined and illustrated as one map in the Map Viewer pane. For more information about this option, see Combining Map Data for Display.

  • Save

    Lets you save selected maps in a single file. For more information about this option, see Saving Map Data.

  • Delete

    Lets you remove the selected map data from the Viewer. When you select this option, graphical images of deleted maps are removed from the Map Viewer pane and map data are removed from the list in the Maps Currently Available pane. For more information about this option, see Removing Map Data.

For information on how to load and manipulate data in this pane, see Loading Map Data.
