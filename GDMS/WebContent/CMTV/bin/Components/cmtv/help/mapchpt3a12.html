Filter Menu

The Filter menu in the Map Viewer window has three options:

  • Hide selected mapped objects

    Lets you remove all selected objects from the maps in the View pane.

  • Hide unselected mapped objects

    Lets you remove all of the objects that are not selected in the View pane.

  • Show all hidden mapped objects

    Lets you restore all previously hidden objects in the View pane.

For more information on these options, see Filtering Maps.
