Saving Your Maps

You can save your maps and comparisons in XML format. For information on saving comparisons, see Saving Comparisons.

To save a single map

    • In the Map window, select File >> Save

The Save window appears.

The navigation options in this window are identical to those in the Open window. For help with these options, see Loading Map Data from a Flatfile.

    • Select a file name

      OR

    • Type in a file name
    • Select Save

To save multiple maps

    • Press and hold <Control> as you select the maps you want to save

You can do this in either the Viewer pane or the Maps Currently Available pane.

Note: In the Maps Currently Available pane, you can also select contiguous maps by clicking and dragging or by selecting the map at the beginning or end of the contiguous block, pressing and holding <Shift>, and selecting the map at the other end of the block.

    • In the Maps Currently Available pane, select Save

To save all maps and comparisons

    • Select File >> Save all to XML

The Save window appears.

The navigation options in this window are identical to those in the Open window. For help with these options, see Loading Map Data from a Flatfile.

    • Select a file name

      OR

    • Type in a file name
    • Select Save
