Saving Comparisons

You can save one or more comparisons in a file for later access using the Save option in the Map Comparisons pane. You can also save all maps and comparisons to a single XML file. For information about that, see To save all maps and comparisons.

To save one or more comparisons

    • Select the comparison(s) you want to save

      Note: To select multiple comparisons, press and hold <Control> while you select them.

    • Select Save

The Save window appears.

This window shows the files currently in the directory shown at the top. The navigation buttons at the top are identical to those in the Open window. For more information about them, see Loading Map Data from a Flatfile.

    • Select a file you want to overwrite or type in a new file name
    • Select Save

The file is saved in the directory shown in the window.
