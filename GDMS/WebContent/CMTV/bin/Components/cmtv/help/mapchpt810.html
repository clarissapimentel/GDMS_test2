Column Dimension

This button shows what kind of data currently appear in the matrix columns. When you select the button, other data options are listed. Depending on how many data dimensions are available for the data type, changing this selection may also change the row dimension.

To change the column contents

    • Select the Column dimension button

A list of data options appears.

    • Select the option you want to appear as columns in the matrix
