Using the Color Definition Methods

Both of the Display Dialog windows provide three tabbed color definition/selection methods:

  • Swatches

    Provides a selectable color panel. You can simply select the color you want here. The small panel on the right shows a history of the colors you've selected during the session.

  • HSB

    Defines colors in terms of Hue, Saturation, and Brightness (HSB). Hue, or color, is defined in degrees (0 - 360); saturation, or purity, is defined by percentage (0 - 100); brightness, or strength, is defined by percentage (0-100). The large color panel shows a range of selectable colors. The content of this panel depends on which of H, S, and B you have selected on the right. You can select a color in the panel or type in HSB values on the right. On the lower right, the RGB (Red, Green, Blue) values of the selected color are provided.

  • RGB

    Defines colors in terms of the amount of red, green, and blue in them. This method provides three color scales, labelled Red, Green, and Blue. You can use the sliders on the scales to change the color mix or type in a number for each, from 0 to 255, to change the mix.

When you select to define colors using any of these methods, the Preview section updates to show you how your selection will look.

To use the Swatches method to select color

    • Select the Swatches tab if it is not already selected
    • Select the color you want in the large color panel

The Recent panel updates to show the last selected color in the upper left corner. When you move your mouse cursor over a color box in this panel, the RGB values for the color are displayed.

    • Look at the Preview section to ensure that your selection is what you want
    • Repeat as needed

To use the HSB method to select color

    • Select the HSB tab if it is not already selected
    • Select the circle to the left of B
    • In the large color block, select in an area that is close to the color you want

The Preview section show the color you've selected.

    • Look at the preview to ensure that your selection is what you want
    • To adjust the color, select in the color block again

      OR

    • Select the H circle and select in the new color block

      OR

    • Select the S circle and select in the new color block
    • Repeat as needed

To use the RGB method to select color

    • Select the RGB tab if it is not already selected
    • Click and drag the slider on each of the scales until you get the color you want

      OR

    • Type in new values for R, G, and B until you get the color you want
