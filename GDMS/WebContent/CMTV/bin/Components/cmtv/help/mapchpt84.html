Using the ISYS Matrix Viewer

The ISYS Matrix Viewer is one of the Viewer services in ISYS. It presents data in rows and columns and provides options you can use to change the data view.

You can easily adjust the width of columns in the Viewer. For help with this, see Adjusting Column Width.

There are three menus at the top of the window:

  • File Menu
  • View Menu
  • Options Menu

There are four buttons at the top of the window.

  • Data Type
  • Row Dimension
  • Column Dimension
  • Display As

To select a row

    • Select the row header at the far left of the window

To select multiple rows

    • Press and hold <Control> as you select the the row headers

      OR

    • Select the row header for the first or last row in a contiguous block
    • Press and hold <Shift> as you select the row header of the row at the other end of the block

To select a column

    • Select the column header at the top of the column

To select multiple columns

    • Press and hold <Control> as you select the column headers

      OR

    • Select the column header for the first or last column in a contiguous block
    • Press and hold <Shift> as you select the column header for the column at the other end of the block
