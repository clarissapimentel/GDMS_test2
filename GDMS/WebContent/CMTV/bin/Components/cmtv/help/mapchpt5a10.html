Customize Layout

This option lets you select the objects that will appear in your graphical maps.

To customize your map display configuration

    • Select View >> Customize layout

The Layout Dialog window appears.

The options here let you define the number and content of panels in your maps. For example, you might want the maps to have a panel for both DNA sequence strands. The main section of the window has options that let you choose the class of object in the panel and the attributes that will be displayed for that object class:

  • Choose an object

    Accesses a selectable list of objects you can select for the panel.

  • ...and its attributes to display

    Select items in this list to define how the objects will appear in the panel.

The window has six button options:

  • Delete

    Lets you delete the selected panel after confirmation.

  • New

    Lets you create a new panel. Each panel has its own tab.

  • OK

    Applies your selections to the currently selected maps and closes the window.

  • Apply to selected

    Applies the currently selected options to any selected maps but leaves the window open so you can create another panel or make changes.

  • Set as default

    Saves the selected options and uses them for all maps that are opened subsequently in the sessions.

  • Cancel

    Closes the window without saving your selections or applying any changes you have made since you last selected Apply to selected.

To define the current panel

    • Select the class of object you want
    • Select the checkboxes for the attributes you want displayed for the object in the panel
    • Select Apply to selected

Your selections are applied immediately to the selected maps.

To add another panel

    • Select New

Another panel tab is added at the top and the new panel is selected.

You can proceed to define the contents of this panel.

To access another panel

    • Select the panel tab

To delete a panel

    • Select the tab for the panel you want to delete
    • Select Delete

A confirmation dialog appears.

    • Select Yes

The tab and the definition within it are deleted. If you deleted one of several panels the remaining panels may be renumbered to retain consecutive numbering.

To apply your layout definition to selected maps

    • Select the maps you want

      For help with this, see Selecting and Deselecting Maps.

    • Select Apply to selected

Your definition is applied to the maps you have selected only. The window remains open and you can make other layout changes.

To set your layout definition as the default

    • Select Set as default

All maps that are displayed from this point on in your session will have the layout you defined. When you exit the Viewer, the layout options are reset to the system default.

To close the Layout Dialog window and apply your selections

    • Select OK

The window closes and your layout selections are applied to any currently selected maps.

To close the Layout Dialog without applying your selections

    • Select Cancel

The window closes. Any changes you've made since you last selected Apply to selected are neither saved nor applied.
