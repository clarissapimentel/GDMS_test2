About ISYS

ISYS is a software platform that lets genomic researchers and bioinformaticians access and use disparate software tools and data sources in a seamless environment. Components that meet ISYS specifications automatically interoperate with other ISYS components. Components can be added or removed--plugged or unplugged--from the platform arbitrarily without impacting other components.

As you use the Comparative Map and Trait Viewer in ISYS, you will be able to access the relevant functionality, or services, of other ISYS components using DynamicDiscovery`, a patent pending feature of ISYS. When you use DynamicDiscovery, ISYS retrieves a list of all of the available and relevant services for the data with which you are working.
