Hiding and Unhiding Comparisons

The Viewer pane can become quite cluttered when you are working with comparisons. Because of this, the Viewer lets you hide and restore, or unhide, comparisons.

To hide comparisons

    • In the Map Viewer window, select Map comparisons >> Hide all map comparisons

All of the comparison lines are removed from the Viewer pane.

To unhide comparisons

    • Select the maps for which you want to see the comparison lines again

For help with selecting maps, see Selecting and Deselecting Maps.

    • In the Map Viewer window, select Map comparisons >> View all comparisons between selected maps

The comparison lines for the selected maps are restored to the Viewer pane.
