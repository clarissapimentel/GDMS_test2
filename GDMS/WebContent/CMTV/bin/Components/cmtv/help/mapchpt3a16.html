Map Viewer Pane

The Map Viewer pane is on the right side of the main window. It is the largest pane--a framework in which one or more maps can be displayed along with map comparisons and consensus maps. This pane is also sometimes called the Desktop.

As shown here, each map within this pane appears in its own window with menu options for that map only:

  • File Menu
  • Edit Menu
  • View Menu
  • Filter Menu
  • Map Comparison Menu

In addition to these five menus, Consensus maps have a Marker Manipulations Menu.

All maps initially appear in the far left portion of the pane. So, if you have more than one map open, you must move the maps to see them. For help with this, see To move a window. You may also need to enlarge the pane to see multiple maps. For help with this, see To resize panes.
