Having the Viewer Compare Selected Maps

You can select two or more maps and have the viewer compare them.

    • In the Viewer or the Maps Currently Available pane, press and hold <Control> as you select the maps you want to compare
    • In the Map Viewer window, select Map comparison >> Generate map comparison for selected maps

The comparison is generated and the panes are updated with comparison information. The comparison is generated using the default algorithm or, if you selected an algorithm, your selection. For more information about selecting an algorithm, see Selecting a Correspondence Algorithm for Your Comparisons.
