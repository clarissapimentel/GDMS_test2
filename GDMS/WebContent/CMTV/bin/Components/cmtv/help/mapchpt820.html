View Menu

The View menu in the Table Viewer has seven options:

  • Hide/Show columns

    Lets you select which columns to show and which to hide.

  • Show all hidden rows

    Lets you make all rows display.

  • Hide selected row(s)

    Lets you have selected data rows removed from the display.

  • Select all rows

    Lets you select all of the rows in the Viewer.

  • Invert selection

    Lets you reverse the current selection, i.e., deselect all currently selected rows while selecting the other rows.

  • Sort ascending by

    Lets you select a column to sort on; the rows will be sorted in ascending order by the selected column.

  • Sort descending by

    Lets you select a column to sort on; the rows will be sorted in descending order by the selected column.

To hide columns

    • Select View >> Hide/Show columns

Another Table Viewer window opens with a listing of the columns in the first window. There is a checkbox to the left of each column name in the list. Only checked columns are shown in the table.

    • Select the checkbox to the left of each column you want to hide to uncheck the box

As each box is unchecked, the column is removed from the first Table Viewer window.

To close the window

    • Select Close

To show previously hidden columns

    • Select View >> Hide/Show columns
    • Select the checkbox for each column you want to restore

      OR

    • Select Show all columns to restore all hidden columns

To close the window

    • Select Close

To hide rows

    • Select the row(s) you want to hide.

      Note: To select multiple rows, press and hold <Control> as you select the rows or, to select a contiguous block of rows, select the first or last row in the block, press and hold <Shift>, and select the row at the other end of the block.

    • Select View >> Hide selected row(s)

The rows are removed from the window.

To restore previously hidden rows to the window

    • Select View >> Show all hidden rows

All rows are shown in the window

To select all rows

    • Select View >> Select all rows

      OR

    • Press and hold <Control> as you press <a>

All rows in the matrix are selected

To invert the selection

    • Select View >> Invert selection

The rows that were previously selected are deselected while all other rows are selected.

To sort the rows in ascending order

    • Select View >> Sort ascending by

A list of all of the column headers appears.

    • Select the column on which you want to sort

The rows are sorted in ascending order by the column you selected.

To sort the rows in descending order

    • Select View >> Sort descending by

A list of all of the column headers appears.

    • Select the column on which you want to sort

The rows are sorted in descending order by the column you selected.
