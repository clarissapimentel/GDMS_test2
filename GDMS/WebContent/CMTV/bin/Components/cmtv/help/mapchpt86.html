View Menu

The View menu in the Matrix Viewer has eight options:

  • Select data type

    Lets you select what type of data will be displayed in the matrix cells, if more than one type is available.

  • Hide/Show columns

    Lets you select which columns to show and which to hide.

  • Show all hidden rows

    Lets you have all rows displayed.

  • Hide selected row(s)

    Lets you have selected data rows removed from the display.

  • Select all rows

    Lets you select all of the rows in the Viewer.

  • Invert selection

    Lets you reverse the selection, i.e., deselect all currently selected rows while selecting the other rows.

  • Sort ascending by

    Lets you select a column to sort by; the rows will be sorted in ascending order by the selected column.

  • Sort descending by

    Lets you select a column to sort by; the rows will be sorted in descending order by the selected column.

To select the data type

    • Select View >> Select data type

      OR

    • Select the Data type button

The Select Data Type window appears.

    • Select the data type you want displayed in the matrix
    • Select OK

To hide columns

    • Select View >> Hide/Show columns

Another Matrix Viewer window opens with a listing of the columns in the first window. There is a checkbox to the left of each column name in the list. Only checked items are shown in the matrix.

    • Select the checkbox to the left of each column you want to hide to uncheck the box

As each box is unchecked, the column is removed from the first Matrix Viewer window.

To close the window

    • Select Close

To show previously hidden columns

    • Select View >> Hide/Show columns
    • Select the checkbox for each column you want to restore

      OR

    • Select Show all columns to restore all hidden columns

To close the window

    • Select Close

To hide rows

    • Select the row(s) you want to hide.

      Note: To select multiple rows, press and hold <Control> as you select the rows or, to select a contiguous block of rows, select the first or last row in the block, press and hold <Shift>, and select the row at the other end of the block.

    • Select View >> Hide selected row(s)

The rows are removed from the window.

To restore previously hidden rows to the window

    • Select View >> Show all hidden rows

All rows are shown in the window

To select all rows

    • Select View >> Select all rows

      OR

    • Press and hold <Control> as you press <a>

All rows in the matrix are selected

To invert the selection

    • Select View >> Invert selection

The rows/columns that were previously selected are deselected while all other rows/columns are selected.

To sort the rows in ascending order

    • Select View >> Sort ascending by

A list of all of the column headers appears.

    • Select the column on which you want to sort

To sort the rows in descending order

    • Select View >> Sort descending by

A list of all of the column headers appears.

    • Select the column on which you want to sort
