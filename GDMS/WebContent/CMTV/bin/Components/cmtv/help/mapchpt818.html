Using the ISYS Table Viewer

The ISYS Table Viewer is one of the Viewer services in ISYS. It presents data in a tabular format.

You can easily adjust the width of columns in the Viewer. For help with this, see Adjusting Column Width.

The Table Viewer has three menus at the top:

  • File
  • View
  • Options

To select a row

    • Select any cell in the row
