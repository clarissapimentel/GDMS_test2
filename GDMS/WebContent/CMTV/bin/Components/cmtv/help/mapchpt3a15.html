Map Comparisons Pane

The Map Comparisons pane is in the lower left corner of the Map Viewer window.

When data are present, this pane provides a summary listing of comparisons and the maps they compare. Each comparison is represented in the listing with a folder icon and a toggle switch:

Selecting the switch will cause the list of maps in the comparison to appear or disappear from the pane.

There are four buttons at the bottom of the pane:

  • View graphically

    Lets you have the selected comparisons illustrated in the Map Viewer pane. For more information about this option, see Viewing Comparisons Graphically.

  • View tabular

    Lets you view a tabular summary of the comparison. For more information about this option, see Viewing Comparisons Tabularly.

  • Save

    Lets you save one or more comparisons to an XML file. For more information about this option, see Saving Comparisons.

  • Delete

    Lets you remove selected comparisons from the Viewer. When you select this option, graphical images of deleted comparisons are removed from the Map Viewer pane and the comparisons are removed from the Map Comparisons pane. For more information about this option, see Deleting Comparisons.
