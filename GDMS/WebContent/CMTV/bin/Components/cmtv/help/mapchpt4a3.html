Loading Map Data from a Database

The procedures and options for loading map data from a database will vary depending on the ISYS services available in your system. You might, for example, be able to retrieve a map based on its identifier. You might also be able to select maps from a list of the maps available in a database. Procedures for these two examples follow; however, your system may not have the options described here.

To retrieve a map by its identifier

    • Select File >> Load

The Load submenu appears.

This menu may vary depending on your ISYS configuration.

    • Select Retrieve by Identifier via ISYS Service

The Select Service Implementations to Execute window appears with a list of available options.

    • Select the option you want
    • Select OK

The Input dialog appears.

    • Type in the identifier for the map you want
    • Select OK

The map data are loaded and the map is listed in the Maps Currently Available pane.

To retrieve a map from a list of maps in a database

    • Select File >> Load

The Load submenu appears.

This menu may vary depending on your ISYS configuration.

    • Select Retrieve list of maps available via ISYS Services

The Select Service Implementations to Execute window appears with a list of available options.

    • Select the option you want
    • Select OK

An ISYS Status dialog appears.

You can select OK to close it. You can also select the checkbox to prevent this dialog from appearing in the future.

Eventually an ISYS Table Viewer window opens with a list of the maps in the selected database.

For information about the Table Viewer window options, see Using the ISYS Table Viewer.

    • Select the map(s) you want to load

      Note: To select more than one map, press and hold <Control> as you select the maps or, to select a contiguous set of maps, click and drag to select maps or select the first or last map in the block you want and press and hold <Shift> as you select the map at the other end of the block

    • Select OK

After a pause, the selected maps are listed in the Maps Currently Available pane.
