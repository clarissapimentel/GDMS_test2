Installing maxdView into ISYS

In order to use the maxdView component, the core classes for maxdView must be obtained from http://www.bioinf.manchester.ac.uk/microarray/maxd/download.html and installed into your ISYS distribution. If you have the Java JDK and are comfortable with command line operations, you can simply download the maxdView JAR file and unpack the contents into ISYS_HOME/bin/Components/maxdView/lib as:

cd ISYS_HOME/bin/Components/maxdView/lib
jar xf /path/to/downloaded/maxdView_version.jar

Alternatively, for a GUI-driven installation, you can download the maxdSetup program. Execute the downloaded JAR file (e.g. by double-clicking it) and select the following path of options in the dialog that appears:

  1. Install or upgrade 'maxdView'
  2. Download latest 'maxdView' from website
  3. Try to connect
  4. Set download options
  5. (Choose a temporary file location and> Commence download
  6. Unpack this file
  7. (into folder: ISYS_HOME/bin/Components/maxdView/lib) and Start unpacking

After you have performed these actions, try clicking the maxdView button in the main ISYS window; it should now launch the maxdView window instead of displaying these instructions.
